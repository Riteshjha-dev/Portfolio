# Personal Portfolio - RITESH JHA

A modern, responsive and fully animated developer portfolio built using **HTML, CSS and JavaScript**.

Designed with a premium UI/UX, smooth animations, custom cursor effects and live GitHub integration, this portfolio showcases my projects, skills and contact information in a clean and interactive way.

---

## 📸 Preview

> A sleek one-page developer portfolio with:
- Animated Hero Section
- Custom Mouse Cursor
- Live GitHub Repositories
- Skills Showcase
- Responsive Design
- Contact Section
- Smooth Scroll Animations

### 💻 Live GitHub Integration

Instead of hardcoding projects, this portfolio fetches the latest public repositories directly from GitHub using the GitHub REST API.

Features include:

- Automatic repository loading
- Dynamic project cards
- Repository language detection
- Repository topics
- Live repository count
- GitHub profile statistics
